# Open Agents Kit（崽崽云）

> 轻量级多 Agent 协作系统 · 5个专业化崽替你打工

---

## 一句话定位

**崽崽云：基于OpenClaw构建的多Agent协作系统，5个专业化AI崽替你打工——研究、编码、调度、整理、审核全自动。**

---

## 核心特性

### 🐣 5个专业崽，分工明确

| 崽 | 职责 | 能做什么 |
|---|---|---|
| **研究崽** researcher_chief | 信息采集与调研 | 多引擎搜索、深度研究、新闻摘要 |
| **码农崽** coder_chief | 代码与工程任务 | GitHub操作、代码审查、浏览器自动化 |
| **调度崽** silijian_chief | 任务调度协调 | 分解任务、分发执行、汇总结果 |
| **整理崽** organizer_chief | 数据整理与输出 | 报告生成、内容归档、知识管理 |
| **审核崽** reviewer_chief | 质量把控 | 审核评估、改进建议、质量评分 |

### 🚀 快速开始

`ash
# 查看崽崽状态
~/oc agents list

# 发送任务
bash ~/oc agent --agent silijian_chief --message 'Your task here'
`
